
## Descrição

Microsserviço de notificações (tipo push) de uma aplicação. Desenvolvida em Node.js com Typescript, Nest.js, Prisma e SQLite. Arquitetura de microsserviços utilizada.

## Instalação

```bash
$ npm install
```

## Rodando a aplicação

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Testes

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
